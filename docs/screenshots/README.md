# Screenshots

Place real product screenshots in this folder before publishing a polished GitHub repository or portfolio page.

Recommended files:

- `dashboard.png`
- `activity-data.png`
- `emission-factors.png`
- `organization-boundary.png`
- `report-page.png`
- `mobile-view.png`

Avoid screenshots containing real customer names, employee names, internal audit evidence, private URLs, or confidential business data.
